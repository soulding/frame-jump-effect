## 💡 关键技术点

### 1. 图层合成原理
```
相框前景层（最上层）
    ↓ 遮挡
人物主体层（中间层，透明背景）
    ↓ 叠加
相框背景层（最下层）

效果：人物仿佛从相框中"跃出"
```

### 2. 手势识别
- **单指**: 拖动（记录初始位置，计算位移）
- **双指**: 缩放 + 旋转（计算距离变化和角度变化）

### 3. 性能优化
- 图片压缩至 1024px 以内
- 渲染节流至 60fps
- 模型单例模式
- 及时释放内存

## 📝 重要提醒

### ⚠️ TensorFlow.js 兼容性

微信小程序**不支持**直接运行 TensorFlow.js，需要：

1. **首选**: 微信 AI 开放平台的人体分割 API
2. **备选**: 云函数 + TF.js 后端处理
3. **备选**: 第三方 AI 插件

详见 `TFJS_ADAPTER.md`

### ⚠️ 相框素材

需要自行设计制作相框 PNG 文件：
- 背景层：完整相框
- 前景层：只有遮挡部分（透明背景）

详见 `FRAMES_GUIDE.md`

### ⚠️ AppID 配置

记得在 `project.config.json` 中填入你的小程序 AppID：
```json
{
  "appid": "your-appid-here"
}
```

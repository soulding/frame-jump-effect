# SBTI 人格测试小程序

一个基于原网站 https://sbti.unun.dev/ 的微信小程序版本。

## 功能特点

- ✅ 完整 31 道测试题目
- ✅ 29 种人格类型匹配
- ✅ 15 维度评分展示
- ✅ 匹配度计算
- ✅ 结果页面可截图保存
- ✅ 纯前端实现，无需后端

## 使用方法

### 1. 导入微信开发者工具

1. 下载并安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 打开微信开发者工具
3. 选择"导入项目"
4. 选择本项目目录 `/root/.openclaw/workspace-dev/sbti-miniprogram`
5. 填写你的小程序 AppID（或使用测试号）

### 2. 修改 AppID

编辑 `project.config.json`，将 `appid` 改为你的小程序 AppID：

```json
{
  "appid": "你的 AppID"
}
```

### 3. 预览和发布

- 在微信开发者工具中点击"预览"扫码测试
- 测试完成后点击"上传"发布版本

## 十五维度

- **S1/S2/S3**: 自我认知、自我清晰、成就动机
- **E1/E2/E3**: 依恋焦虑、投入程度、独立需求
- **A1/A2/A3**: 人性看法、规则态度、意义追求
- **Ac1/Ac2/Ac3**: 趋避倾向、决策风格、执行风格
- **So1/So2/So3**: 社交倾向、亲密需求、社交面具

## 技术实现

- 人格匹配算法基于 15 维度的 H/M/L 等级匹配
- 每个维度 2 道题，满分 6 分
- 2.5 分以上为 H (High)，1.5 分以下为 L (Low)，中间为 M (Medium)
- 匹配度 = 匹配的等级数 / 总等级数 (15) × 100%

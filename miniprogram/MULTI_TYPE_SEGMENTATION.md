# 多类型 AI 分割支持

## 🎯 支持的分割类型

现在小程序支持**9 种**不同的 AI 分割类型，不再局限于人像！

### 分割类型对比

| 类型 | 图标 | 精度 | 速度 | 适用场景 | API 支持 |
|------|------|------|------|----------|---------|
| **人像** | 👤 | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ | 人物照片、自拍、合影 | ✅ |
| **人脸** | 😊 | ⭐⭐⭐⭐⭐ | ⭐⭐⭐⭐ | 人脸特写、证件照 | ✅ |
| **宠物** | 🐶 | ⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ | 猫、狗等宠物 | ✅ |
| **动物** | 🦁 | ⭐⭐⭐ | ⭐⭐⭐ | 野生动物、鸟类等 | ⚠️ |
| **植物** | 🌿 | ⭐⭐⭐ | ⭐⭐ | 花卉、树木、盆栽 | ⚠️ |
| **通用物体** | 📦 | ⭐⭐⭐ | ⭐⭐⭐ | 日常物品、商品 | ⚠️ |
| **车辆** | 🚗 | ⭐⭐⭐⭐ | ⭐⭐⭐⭐⭐ | 汽车、摩托车 | ✅ |
| **食物** | 🍔 | ⭐⭐⭐⭐ | ⭐⭐⭐⭐ | 美食、食材 | ✅ |
| **手动抠图** | ✏️ | ⭐⭐⭐⭐⭐ | ⭐ | 精细抠图、复杂场景 | 待开发 |

**精度说明**:
- ⭐⭐⭐⭐⭐ 极高 - 发丝级精度
- ⭐⭐⭐⭐ 高 - 边缘清晰
- ⭐⭐⭐ 中 - 基本可用
- ⭐⭐ 低 - 需要手动修正

## 🚀 使用方式

### 1. 选择分割类型

在小程序界面点击 **"选择类型"** 按钮，会弹出类型选择器：

```
👤 人像抠图
😊 人脸抠图
🐶 宠物抠图
🦁 动物抠图
🌿 植物抠图
📦 通用物体
🚗 车辆抠图
🍔 食物抠图
✏️ 手动抠图
```

### 2. 选择照片

点击 **"选择照片"**，从相册选择要处理的照片。

### 3. 自动处理

AI 会根据选择的类型自动识别并抠图：
- 人像 → 识别人体
- 宠物 → 识别猫狗
- 植物 → 识别花草
- 等等...

### 4. 调整与保存

- 拖动调整位置
- 双指缩放和旋转
- 点击保存

## ⚠️ 限制与注意事项

### 微信 AI 开放平台支持

不同 AI 服务支持不同类型：

| 服务商 | 支持类型 |
|--------|---------|
| 腾讯优图 | 人像、人脸、宠物、车辆 |
| 百度 AI | 人像、动物、植物、物体 |
| 阿里 AI | 人像、商品、车辆、食物 |
| 商汤科技 | 人像、人脸、宠物 |

**建议**: 优先使用人像、宠物、车辆等成熟类型。

### 性能考虑

不同类型处理时间不同：

```
人像：500-1000ms   ⭐⭐⭐⭐⭐
宠物：500-1000ms   ⭐⭐⭐⭐⭐
车辆：500-800ms    ⭐⭐⭐⭐⭐
食物：800-1200ms   ⭐⭐⭐⭐
动物：1000-2000ms  ⭐⭐⭐
植物：1500-2500ms  ⭐⭐
物体：1000-2000ms  ⭐⭐⭐
```

### 精度影响因素

1. **图片质量**: 清晰度高 → 精度高
2. **背景复杂度**: 简洁背景 → 识别准确
3. **光线条件**: 充足光线 → 边缘清晰
4. **物体大小**: 主体突出 → 效果好

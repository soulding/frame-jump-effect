# 相框素材制作指南

## 素材规格

### 尺寸要求

| 类型 | 建议尺寸 | 格式 | 说明 |
|------|---------|------|------|
| 背景层 | 750x1000px | PNG | 2x 分辨率，适配高清屏 |
| 前景层 | 750x1000px | PNG | 透明背景，只有遮挡部分 |

### 设计要求

#### 背景层 (bg.png)
- 完整的相框设计
- 中间留空区域用于显示人物
- 可以是任何风格（经典、现代、复古等）
- 背景色建议为纯色或简单渐变

#### 前景层 (fg.png)
- **只有遮挡部分**，其余区域完全透明
- 通常包括：顶部装饰、底部装饰、边角装饰
- 用于制造"人物被相框卡住"的立体效果
- 透明区域 alpha = 0

## 示例结构

```
经典相框示例:

┌─────────────────────────────┐
│  ════════════════════════   │ ← 前景层（顶部装饰）
│  │                   │      │
│  │                   │      │
│  │    人物区域        │      │ ← 透明，显示抠图人物
│  │                   │      │
│  │                   │      │
│  ════════════════════════   │ ← 前景层（底部装饰）
└─────────────────────────────┘
     ↑
  背景层（完整相框）
```

## 制作工具推荐

### 专业工具
- **Adobe Photoshop**: 图层 + 蒙版
- **Affinity Photo**: 性价比高的替代品
- **GIMP**: 免费开源

### 在线工具
- **Photopea**: 在线 Photoshop 替代
- **Canva**: 模板化设计
- **Figma**: 矢量设计

## 制作步骤（Photoshop）

### 1. 创建背景层

```
1. 新建画布：750x1000px，透明背景
2. 设计相框边框和装饰
3. 中间区域保持透明（人物显示区）
4. 导出为 classic_bg.png
```

### 2. 创建前景层

```
1. 复制背景层
2. 删除中间人物区域的内容
3. 只保留需要遮挡人物的部分
4. 确保其余区域完全透明
5. 导出为 classic_fg.png
```

### 3. 测试效果

```
1. 导入一张人物照片
2. 按顺序叠放：背景层 → 人物 → 前景层
3. 检查遮挡效果是否自然
4. 调整前景层位置直到满意
```

## 相框配置

在 `utils/frame-composer.js` 中添加新相框配置：

```javascript
vintage: {
  id: 'vintage',
  name: '复古相框',
  background: {
    url: '/miniprogram/frames/vintage_bg.png',
    width: 375,
    height: 500
  },
  foreground: {
    url: '/miniprogram/frames/vintage_fg.png',
    width: 375,
    height: 500,
    occlusionZones: [
      { x: 40, y: 40, width: 295, height: 120 },
      { x: 40, y: 380, width: 295, height: 120 }
    ]
  },
  subjectPosition: {
    x: 0,
    y: 0,
    scale: 1.0
  },
  allowedGestures: ['drag', 'pinch', 'rotate']
}
```

## 设计技巧

### 制造立体感
1. **顶部遮挡**: 相框顶部装饰压住人物头顶
2. **底部遮挡**: 相框底部装饰压住人物身体下部
3. **侧边装饰**: 可选，增加层次感

### 风格建议
- **经典**: 木质纹理、金色花纹
- **现代**: 简洁线条、几何图形
- **复古**: 做旧效果、怀旧色彩
- **创意**: 特殊形状、主题装饰

### 注意事项
- 前景层透明区域要干净，不要有残留像素
- 遮挡边缘要自然，避免生硬切割
- 考虑不同身高人物的适配
- 测试多张照片确保效果一致

## 测试验证

制作完成后，在小程序中测试：

1. 导入素材到 `miniprogram/frames/`
2. 编译运行小程序
3. 选择测试照片
4. 检查相框显示是否正常
5. 调整人物位置测试遮挡效果
6. 保存并分享验证最终输出

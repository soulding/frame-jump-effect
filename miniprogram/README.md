# 跃出相框 - 微信小程序

🖼️ 实现照片主体从相框中"跃出"的 3D 视觉效果

## 效果演示

通过图像分层技术和 AI 人体分割，利用前景遮挡关系创造物体正向屏幕外突出的立体感。

```
用户照片 → AI 抠图 → 透明背景 PNG → 叠加相框图层 → 前景遮挡合成 → 最终输出
```

## 技术栈

- **平台**: 微信小程序
- **前端**: JavaScript (ES6+)
- **渲染**: Canvas 2D API
- **AI**: TensorFlow.js + BodyPix 模型（纯前端）

## 快速开始

### 1. 准备工作

1. 下载并安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)
2. 注册微信小程序账号，获取 AppID
3. 修改 `project.config.json` 中的 `appid` 字段

### 2. 导入项目

1. 打开微信开发者工具
2. 选择"导入项目"
3. 选择项目目录：`miniprogram/`
4. 填入你的 AppID
5. 点击"导入"

### 3. 添加相框素材

在 `miniprogram/frames/` 目录下添加相框图片：

- **背景层** (`*_bg.png`): 完整的相框背景图
- **前景层** (`*_fg.png`): 只有遮挡部分的透明 PNG

建议尺寸：750x1000px（2x 分辨率）

### 4. 配置 BodyPix 模型

由于微信小程序环境限制，需要：

1. 下载 BodyPix 模型文件到本地
2. 或使用兼容小程序的 AI 服务
3. 修改 `utils/bodypix-loader.js` 中的模型加载逻辑

### 5. 运行调试

1. 在开发者工具中点击"编译"
2. 在模拟器或真机上测试
3. 检查控制台日志

## 核心功能

### 图像分割流程

```javascript
1. 用户选择照片
2. 压缩图片（优化性能）
3. 加载 BodyPix 模型
4. 执行人体/物体分割
5. 生成透明背景 PNG
6. 输出抠图结果
```

### 图层合成（从底到顶）

```
┌────────────────────────────┐
│ 第 3 层：相框前景层          │ ← 最上层，制造"被框卡住"效果
├────────────────────────────┤
│ 第 2 层：抠出主体 PNG        │ ← 带透明通道的主体图像
├────────────────────────────┤
│ 第 1 层：相框背景层          │ ← 基础相框图层
└────────────────────────────┘
```

### 手势交互

- **单指拖动**: 移动主体位置
- **双指缩放**: 调整主体大小（0.5x - 1.5x）
- **双指旋转**: 调整主体角度（-30° - 30°）

## 注意事项

### ⚠️ 微信小程序环境限制

1. **TensorFlow.js 兼容性**: 
   - 小程序不支持直接加载 CDN 的 TF.js
   - 需要使用小程序专用版本或云函数
   
2. **替代方案**:
   - 使用微信 AI 开放平台的人体分割能力
   - 使用云函数调用后端 AI 服务
   - 使用第三方小程序 AI 插件

3. **Canvas 限制**:
   - 小程序 Canvas 2D 接口与 Web 有差异
   - 需要使用 `wx.createImage()` 加载图片
   - 注意异步加载和绘制时机

### 推荐实施方案

对于生产环境，建议：

1. **使用微信 AI 能力**:
   ```javascript
   // 微信人体分割 API
   wx.ai.segmentPerson({
     imagePath: tempFilePath,
     success: (res) => {
       // 获取分割掩码
     }
   })
   ```

2. **使用云函数**:
   - 在云函数中调用 TensorFlow.js
   - 返回处理后的透明图片
   - 前端只负责合成和交互

3. **使用第三方插件**:
   - 搜索微信小程序 AI 插件市场
   - 选择人体分割类插件
   - 按插件文档集成

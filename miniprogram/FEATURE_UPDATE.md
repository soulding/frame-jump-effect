## ⚠️ 注意事项

### 精度对比
- **最高精度**: 人像、人脸（发丝级）
- **高精度**: 宠物、车辆、食物
- **中等精度**: 动物、植物、物体

### 处理速度
- **最快**: 人像、宠物、车辆 (< 1 秒)
- **中等**: 食物、物体 (1-2 秒)
- **较慢**: 植物、动物 (2-3 秒)

### 最佳实践
1. **清晰照片**: 高清晰度 → 高精度
2. **简洁背景**: 背景简单 → 识别准确
3. **充足光线**: 光线好 → 边缘清晰
4. **主体突出**: 物体占比大 → 效果好

export const SegmentType = {
  PERSON: 'person',
  PET: 'pet',

### 调用方式
```javascript
// 选择类型
this.setData({ segmentType: 'pet' });

// 调用分割

  model,
  imagePath,
  width,
  height,
  'pet'  // 类型参数
);
```

```javascript
wx.cloud.callFunction({
  name: 'segment-person',

  }

```
